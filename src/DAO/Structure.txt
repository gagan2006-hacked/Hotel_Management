|   Service (Java File)   |   Repository (DAO File)         |   Main Table      |   What It Does                                                         |
| ----------------------- | ------------------------------- | ----------------- | ---------------------------------------------------------------------- |
| `RoomService.java`      | `RoomRepository.java`           | `room`            | Add, update, delete rooms; change room status; find available rooms    |
| `CustomerService.java`  | `CustomerRepository.java`       | `customer`        | Add customers, view details, search by name/email, get booking history |
| `StaffService.java`     | `StaffRepository.java`          | `staff`           | Manage staff (add/remove/update), filter by role/status                |
| `BookingService.java`   | `BookingRepository.java`        | `booking`         | Create, modify, cancel, or view bookings                               |
| `PaymentService.java`   | `PaymentRepository.java`        | `payment`         | Record payments, update status, get payments by booking                |
| `ReportService.java`    | `BusinessReportRepository.java` | `business_report` | Generate and store monthly profit/loss reports, calculate growth       |


| **Repository (DAO File)**           | **Main Table**            | **Core Methods (with brief purpose)**                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| ----------------------------------- | ------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **`RoomRepository.java`**           | `room`                    | - `addRoom(Room room)` → Insert new room record. - `updateRoom(Room room)` → Update room details (price, type, etc.).<br> - ` deleteRoom(int roomId) ` → Remove a room from DB.<br> - `getRoomById(int roomId)` → Fetch room info by ID.<br> - `getAllRooms()` → List all rooms.<br> - `getAvailableRooms()` → Get rooms where status = 'available'.<br> - `updateRoomStatus(int roomId, String status)` → Change room availability.                                                                                                                                  |
| **`CustomerRepository.java`**       | `customer`                | - `addCustomer(Customer customer)` → Insert new customer record. - `updateCustomer(User customer)` → Edit name, phone, or email.<br> - `getCustomerById(int id)` → Fetch specific customer.<br> - `getCustomerByEmail(String email)` → Retrieve customer by email.<br> - `getAllCustomers()` → Get all customers.<br> - ` getCustomerBookingHistory(int userId) ` → Join bookings table to get user’s history.                                                                                                                                                          |
| **`StaffRepository.java`**          | `staff`                   | - `addStaff(Staff staff)` → Add new staff. - `updateStaff(Staff staff)` → Edit staff details.<br> - `deleteStaff(int staffId)` → Remove staff.<br> - `getStaffById(int staffId)` → View staff info.<br> - `getAllStaff()` → List all staff.<br> - `getStaffByRole(String role)` → Filter by position (manager, cleaner, etc.).<br> - `updateStaffStatus(int staffId, String status)` → Update active/inactive state.                                                                                                                                                  |
| **`BookingRepository.java`**        | `booking`                 | - `createBooking(Booking booking)` → Insert booking record. - `updateBooking(Booking booking)` → Modify dates, guests, etc.<br> - `cancelBooking(int bookingId)` → Update booking status to 'cancelled'.<br> - `getBookingById(int bookingId)` → Fetch single booking.<br> - `getBookingsByUser(int userId)` → Fetch all bookings by user.<br> - `getAllBookings()` → List all bookings.<br> - `updateBookingStatus(int bookingId, String status)` → Confirm, check-in, check-out.<br> - `getActiveBookings()` → Retrieve ongoing bookings.                           |
| **`PaymentRepository.java`**        | `payment`                 | - `addPayment(Payment payment)` → Record a new payment. - `updatePaymentStatus(int paymentId, String status)` → Change payment state.<br> - `getPaymentById(int paymentId)` → Fetch payment details.<br> - `getPaymentsByBooking(int bookingId)` → Retrieve payments related to a booking.<br> - `getAllPayments()` → List all payments.<br> - `getPaymentsByStatus(String status)` → Filter by pending/completed.                                                                                                                                                    |
| **`BusinessReportRepository.java`** | `business_report`         | - `addMonthlyReport(BusinessReport report)` → Insert report data for a month. - `updateReport(BusinessReport report)` → Edit revenue or expense.<br> - `getReportByMonth(int year, String month)` → Fetch a specific month’s report.<br> - `getAllReports()` → Retrieve all business reports.<br> - `calculateProfitLoss()` → Compute totals from payments and expenses.<br> - `calculateGrowthRate(int currentYear, String currentMonth)` → Compare with previous month.<br> - `getAnnualSummary(int year)` → Show total revenue, expenses, and profit for the year. |
