| Service File (Java Class)   |Contains Business Logic For   | **Key Methods / Responsibilities**                                                                                                                                                                                                                                                                                     |
|-----------------------------|------------------------------| ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 🏨 RoomService.java         |   Room Management            | - `addRoom(Room room)` → Add new room<br> - `updateRoom(Room room)` → Edit room details<br> - `deleteRoom(int roomId)` → Remove room<br> - `getAllRooms()` → List all rooms<br> - `updateRoomStatus(int roomId, String status)` → Mark available/booked/maintenance                                                    |
| 👥 CustomerService.java     |   Customer Management        | - `addCustomer(Customer customer)` → Add new customer<br> - `getCustomerById(int id)` → Retrieve details<br> - `getAllCustomers()` → List all customers<br> - `getCustomerBookingHistory(int customerId)` → List bookings for that customer                                                                            |
| 👔 StaffService.java        |   Staff Management           | - `addStaff(Staff staff)` → Add staff member<br> - `updateStaff(Staff staff)` → Update salary/role/status<br> - `removeStaff(int staffId)` → Delete staff<br> - `getAllStaff()` → List all staff members                                                                                                               |
| 📅 BookingService.java      |   Booking Management         | - `createBooking(Booking booking)` → Check room availability & create booking<br> - `modifyBooking(Booking booking)` → Update dates or room<br> - `cancelBooking(int bookingId)` → Cancel booking<br> - `completeBooking(int bookingId)` → Mark booking completed & free room<br> - `getBookingDetails(int bookingId)` |
| 💳 PaymentService.java      |   Payment Processing         | - `processPayment(Payment payment)` → Record and validate payments<br> - `updatePaymentStatus(int paymentId, String status)` → Set pending/completed<br> - `getPaymentsByBooking(int bookingId)` → Retrieve payment history<br> - Automatically update booking status to “confirmed” when completed                    |
| 📊 ReportService.java       |   Report Generation          | - `generateMonthlyReport(int year, String month)` → Compute revenue, expenses, profit/loss<br> - `calculateGrowthRate()` → Compare with previous month<br> - `getAllReports()` → Retrieve all reports<br> - Optionally `exportReportToPDF()` / `exportReportToExcel()`                                                 |
